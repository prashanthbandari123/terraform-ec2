# terraform-ec2
Terraform EC2 practice

 EC2 Instance Creation (Manual & Terraform)

## Objective
To launch an AWS EC2 instance manually using AWS Console and automatically using Terraform.

## Steps Performed
1. Created an EC2 instance manually through the AWS Management Console.
2. Installed and configured Terraform and AWS CLI on Windows.
3. Wrote Terraform configuration (`main.tf`) for EC2 provisioning.
4. Executed `terraform init`, `terraform plan`, and `terraform apply`.
5. Verified the EC2 instance in AWS Console and destroyed it using `terraform destroy`.

## Result
EC2 instances were successfully created using both manual and Terraform methods.
