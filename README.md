# Terraform EC2 Instance Creation

## Objective
To provision an AWS EC2 instance using Terraform.

## Steps Performed
1. Installed Terraform on Windows and verified using `terraform version`.
2. Configured AWS credentials using `aws configure`.
3. Created a Terraform project folder and added `main.tf`.
4. Defined AWS provider and EC2 instance resource in `main.tf`.
5. Initialized Terraform using `terraform init`.
6. Checked execution plan using `terraform plan`.
7. Created the EC2 instance using `terraform apply`.
8. Verified the EC2 instance in AWS Console.
9. Destroyed the instance after completion using `terraform destroy`.

## Result
An EC2 instance was successfully created in AWS using Terraform.
